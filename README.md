## 核心创新

**不是健康监测设备，是一个会进化的"数字健康分身"**

```
传统设备：采集数据 → 展示数字 → 等待用户查看
VitaEdge：  数据 → Agent感知 → 主动关心 → 越用越懂你
```

## 技术亮点

| 亮点 | 说明 |
|------|------|
| TinyML on ESP32 | 12KB模型跑在$3芯片上，不是云端 |
| 24h连续监测 | PPG + ECG 双传感器 |
| 本地AI推理 | 延迟<100ms，不需要联网 |
| OpenClaw Agent | 上下文记忆 + 心跳机制 + 情商感知 |
| 家人守护 | 分级预警 + SOS紧急救援 |
